# Figures And Videos

Media assets used by the repository README and tutorials.

- `Proposed_method.png`: overview of the proposed framework.
- `interactive_simulator.png`: browser-based manual packing simulator with a
  Three.js container view and Plotly buffer panel.
- `random_stable_loading_acc.gif`: three random stable loading sequences on the
  real platform, used to support the structural stability validation method.
- `policy_rearrangement_demo.gif`: robot palletizing procedure showing packing,
  unpacking, and repacking with the trained policy and repacking planner.
